NC-Games API

My first backend project was created using Node Express and PostgreSQL. With the MVC model in mind I created a REST API that was seeded with dummy data. Unit and Integration testing was done with the Jest library. Creating this API has given me a great understanding of how REST API's function and a great appreciation for clear and precise data structures.

NC-Games React App

Using the NC-Games API, I created a react app that can interact with that data. This 5 day project utilises React hooks and the Context API to achieve this. My CRUD methods were achieved using AXIOS. I found React to be a challenging and very enjoyable programmatic framework to use and this project also gave me chance to experiment with some fun and quirky CSS styles.
